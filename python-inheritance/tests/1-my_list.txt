-----------------------
How to use 1-my_list.py
-----------------------

----------------------------------------------
Importing
-----------------------------------------------
    >>> MyList = __import__('1-my_list').MyList

``MyList.print_sorted()``
Prints integers in order


-----------
Tests
-----------
   
Prints without list
--------------------------------------
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> print(my_list)
	[1, 4, 2, 3, 5]

Negative
-------------------------------
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(-4)
	>>> my_list.append(-2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> my_list.print_sorted()
	[-4, -2, 1, 3, 5]

Single elements
--------------------------------
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.print_sorted()
	[1]

In order
--------------------------------
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(4)
	>>> my_list.append(5)
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

Empty 
------------------------------
	>>> my_list = MyList()
	>>> my_list.print_sorted()
	[]

Invalid Arguments
=================

' String '
-----------------------------
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append("hello")
	>>> my_list.print_sorted()
	Traceback (most recent call last):
	...
	TypeError
